\section{Generalities on Lipschitz Functions}

\subsection{Lipschitz functions on manifolds and subdifferentials}

\subsubsection{General facts and definitions}

\quad For all the following, $M$ will denote a differentiable manifold of dimension $m$.
In this section, we generalize many definitions related to Lipschitz functions. We begin by
extending the definition of Lipschitz functions to functions on a manifold.

\begin{defi}
    A function $f : M \rightarrow \mathbb{R}^n$, where $M$ is a smooth manifold, is
    locally Lipschitz if for any chart $U \subset M$ and any compact $K \subset U$,
    there exists $C > 0$ such that, with the coordinates on $U$
    \[\forall x, y \in K, | f(x) - f(y) | \leq C |x - y| \]
\end{defi}

\quad Let $f : U \subset \mathbb{R}^m \rightarrow \mathbb{R}^n$ be a Lipschitz function, where $U$ is
open. Rademacher's theorem, which states that $f$ is differentiable almost everywhere, will
allow us to generalize differentials to this setting.

\begin{defi}\cite{ClarkeImplicit}
    The subdifferential of $f$ at $x$, denoted by $\partial_x f$ is the convex hull of
    the linear maps $l$ obtained as
    \[ l = \lim_{i \rightarrow \infty} d_{x_i}f \]
    where $f$ is differentiable at each $x_i$ and $x_i \rightarrow x$.
\end{defi}

The subdifferential is then set valued. This set is always non empty and compact, $f$ being
Lipschitz bounds $df$ where it is defined.\newline
By \cite{ClarkeFirst}, we can exclude a set of measure $0$ from the possible values of $x_i$
without changing the definition.

\begin{prop}
    $\partial f$ is the smallest convex valued upper semi-continuous function containing
    $df$ whenever it is defined.
\end{prop}

\begin{proof}
    First, $\partial f$ is upper semi-continuous : by compactness, it suffices to
    show that for all $x \in U, \varepsilon > 0$, for $y$ sufficiently close to $x$.
    \[ \partial_y f \subset \partial_x f + B(0, \varepsilon)\]
    Assume for contradiction that there exists a sequence $x_i \rightarrow x$ such that for all $i$,
    $\partial_{x_i} f$ has an element out of $A := \partial_x f + \overline{B}(0, \varepsilon/2)$.
    $A$ being convex and closed, let $x'_i$ be such that $d_{x'_i} f$ is out of
    $A$, and such that $x'_i - x_i \rightarrow 0$. Up to extraction, $d_{x'_i} f$ converges to an element in 
    $\partial_x f$, so it enters $A$, contradiction.
    \medskip \newline
    The minimality of $\partial f$ follows from the fact that it is a convex hull.
\end{proof}

\quad The subdifferential allows us to generalize the idea of regular and critical points, which will
be an important tool for generalizing Morse theory.

\begin{defi}
    A point $x \in U$ will be called critical if there exists some $l \in \partial_x f$
    not of maximal rank. $x$ will be called regular if it isn't critical.
\end{defi}

To extend these subdifferentiability notions to manifolds, one must ensure that for all
charts, the definitions of $\partial f$ coincide. If this is the case, $\partial_x f$ is
a compact convex subset of $T_x^* M$. The following generalization of the chain rule
allows us to do that.

\begin{prop}[Subdifferential chain rule]
    Let $h : \mathbb{R}^n \rightarrow \mathbb{R}^k$ be a $C^1$ function. For $x \in U$ :
    \[ \partial_x (h \circ f) \subset d_{f(x)} h \circ \partial_x f \]
    If $h$ is a diffeomorpism, we have equality.
\end{prop}

\begin{proof}
    For the first part, because $f$ is differentiable a.e, it suffices to show that
    the leftwise term is an upper semi-continuous convex valued function that contains
    $d(h \circ f)$ whenever $df$ is defined. This follows from the definition of $\partial f$.
    \medskip \newline
    If $h$ is a diffeomorphism, then for $x \in U$ :
    \[ \partial_x f = \partial_x (h^{-1} \circ h \circ f)
        \subset d_{h \circ f (x)} h^{-1} \circ \partial_x (h \circ f) \subset \partial_x f \]
    so we have equalities, and the result follows.
\end{proof}

Although some important differentiablity related results can be obtained with locally
Lipschitz functions as we'll see, a general locally Lipschitz function can have some
pathological behaviours. As an example, it's subdifferential isn't in general a
topological submanifold of $M$ : define
\[ A = \bigcup_{n \geq 0} \left[ \frac{1}{2^{2n+1}}, \frac{1}{2^{2n}} \right[,
B = \bigcup_{n \geq 0} \left[ \frac{1}{2^{2n+2}}, \frac{1}{2^{2n+1}} \right[ \]
The integral of the following function constitutes a counterexample, no chart can be
found around the point $(0, 1)$ which is a point in the subdifferential.
\[ \begin{array}{cl}
    \mathbb{R} &\longrightarrow \mathbb{R}\\
    x &\longmapsto \left\{ \begin{array}{cl}
        2 & x \in A\\
        1 & x \in B\\
        0 & x \leq 0 \text{ or } x \geq 1
    \end{array} \right.
\end{array} \]
However, the functions we'll
study, for example continuous selections (see Section \ref{sec:ContSelec}) have much nicer
behaviour and we retrieve this property.

\subsubsection{The case of convex functions}\label{sec:subdiffconvex}

\quad If we consider convex Lipschitz functions $f : U \subset \mathbb{R}^m \rightarrow \mathbb{R}$,
where $U$ is a convex open set, the subdifferential can be understood much more precisely
(in fact, this object was first invented for convex functions).
\medskip \newline
Consider a point $x \in U$ where $d_x f$ exists. By convexity, we have
\[ \forall z \in U, f(z) \geq f(x) + d_x f(z-x) \]
Denote $C_x$ the set of all linear maps verifying such property at a point $x$.
The function $C : x \mapsto C_x$ is convex-valued, upper semi-continuous and contains
$d_x f$ whenever it is defined. Moreover, at such points we have for $h$ small enough :
\[ f(x+h) = f(x) + d_x f(h) + o(h) \geq f(x) + \varphi(h) \]
where $\varphi \in C_x$. We deduce $(\varphi - d_x f)(h) = o(h)$, so $\varphi = d_x f$.
Upper semi-continuity gives us $C \subset \partial f$, and the reciprocal is given
by minimality of $\partial f$. So for a convex function the subdifferential can be described
as
\[ \partial_x f = \{ \varphi \in (\mathbb{R}^m)^* \mid \forall z \in U, f(z) \geq f(x) + \varphi(z-x) \}\]
which gives a geometrical meaning to the name subdifferential.
\bigskip \newline
\quad Lastly, we connect the subdifferential to the directionnal derivatives of $f$. The
directionnal derivative of $f$ at $x$ in direction $v \in \mathbb{R}^m$ is given by
\[ D_x f(v) = \lim_{t \rightarrow 0^+} \frac{f(x+tv)-f(x)}{t} \]
and is always defined since $f$ is convex. The two concepts are related by the following
formula
\[ D_x f(v) = \max_{l \in \partial f} \langle l, v \rangle \]

\subsection{Generalized differential theorems}

\quad We now consider the specific case of $f : U \subset \mathbb{R}^m \rightarrow \mathbb{R}^m$.
This section is devoted to proving generalizations by Clarke \cite{ClarkeImplicit} of the inverse function
theorem and the implicit function theorem for the case of Lipschitz functions.
\bigskip

\quad In this section, we norm the space of linear maps $\mathbb{R}^m \rightarrow \mathbb{R}^m$, which we denote
$\mathcal{L}(\mathbb{R}^n)$, with the operator norm $\| \cdot \|$ obtained from the norm used on
$\mathbb{R}^n$.

\begin{lem}\label{invfunlem}
    Assume $U$ is convex, and that there exists some closed convex set $C$ such that
    \begin{itemize}
        \item Every $l \in C$ is invertible.
        \item $\partial_x f \subset C$ for all $x \in U$
    \end{itemize}
    Then there exists some $\delta > 0$ such that for all $x, y \in U$ :
    \[ |f(x) - f(y)| \geq \delta |x - y| \]
\end{lem}

\begin{proof}
    Firstly, we set
    \[ \delta = \inf\left\{ |l(h)| \mid l \in C, h \in S \right\} \]
    where $S$ is the unit sphere in $\mathbb{R}^n$. $\delta > 0$ because all $l \in C$
    are invertible and $C \times S$ is closed.
    \medskip \newline
    Let $x, y \in U$ and assume $x \neq y$. Let $\pi$ be the plane perpendicular to $y - x$
    passing through $x$. $f$ being differentiable a.e, Fubini's theorem implies that
    for almost all $x' \in \pi$, $f$ is differentiable almost on the whole line passing
    through $x'$ and perpendicular to $\pi$ (i.e $f$ isn't differentiable on a subset of this line of
    one dimensional measure $0$).\newline
    Denoting $F : t \longmapsto f(x' + t(y-x))$, we have :
    \begin{equation}\label{eq:invfunlemeq}
        f(x' + y - x) - f(x') = \int_{0}^{1} F'(t) dt = \int_{0}^1 d_{x'+t(y-x)} f (y-x)dt
    \end{equation}
    Define the linear map
    \[ L = \int_0^1 d_{x'+t(y-x)} f dt \]
    $C$ being convex and closed, $L \in C$. Observe that the integral on the right of
    \eqref{eq:invfunlemeq} is just $L(y-x)$. So
    \[ |f(x'+y-x) - f(x')| \geq \delta |y-x| \]
    This inequality holds for almost all $x' \in \pi$, so by continuity of $f$, it holds for $x$
    and we get the desired result.
\end{proof}

\begin{theorem}[Lipschitz inverse function theorem]
    If $x_0$ is a regular point of $f$, then there exists $V, W$, neighbourhoods of $x_0$ and $f(x_0)$
    respectively, such that $f$ induces a bijection $f : V \rightarrow W$ where $f^{-1}$ is
    Lipschitz too.
\end{theorem}

\begin{proof}
    By compactness of $\partial_{x_0} f$, regularity of $x_0$, and the fact that the invertible
    linear maps form an open set, there exist some $\varepsilon > 0$ such that the set
    \[ C := \partial_{x_0} f + \overline{B}(0, \varepsilon) \]
    is composed only of invertible maps. By upper semi-continuity of $\partial f$, there is
    some $r > 0$ such that for $y \in B(x_0, r)$, $\partial f(y) \subset C$. We then
    apply Lemma \ref{invfunlem} with $U = B(x_0, r)$ and get a corresponding $\delta > 0$.
    \medskip \newline
    Set $V = B(x_0, r/2)$ and $W = f(V)$. We showed in particular that $f$ is injective on
    $\overline{V}$, by compactness $f : \overline{V} \rightarrow f(\overline{V})$ induces a homeomorphism. $W$
    is then open. The inequality we got with $\delta$ implies that $f^{-1}$ is $\frac{1}{\delta}$
    Lipschitz.
\end{proof}

From this theorem, follows the implicit function theorem for Lipschitz functions

\begin{theorem}
    Let $F : U \rightarrow \mathbb{R}^n$, where $U \subset \mathbb{R}^m \times \mathbb{R}^n$
    is open and $F$ is Lipschitz. Let $(x_0, y_0) \in U$ such that $F(x_0, y_0) = 0$
    and $y_0$ is a regular point of $y \mapsto F(x_0, y)$. Then there exist neighbourhoods
    $(x_0, y_0) \in W \subset U$, $x_0 \in V \subset R^m$ and a Lipschitz function
    $g : V \rightarrow \mathbb{R}^n$ such that
    \[ (x, y) \in W \text{ and } F(x, y) = 0 \Leftrightarrow x \in V \text{ and } y = g(x) \] 
\end{theorem}

The proof is almost the same as the usual implicit function theorem, where the continuity
of the differential is replaced by the semi continuity of $\partial F$.

\subsection{First Morse lemma for locally Lipschitz functions}

\quad The goal of this section is to use the generalized notion of critical points in order to
generalize the first Morse lemma to the case of locally Lipschitz functions. Fix $M$
an $m$-dimensional smooth manifold, $f : M \rightarrow \mathbb{R}$ a continuous function and
define for all $c \in \mathbb{R}$ the sublevel set of $f$ of value $c$ as
\[ M_c := f^{-1}(]-\infty, c]) \]
The classical first Morse lemma states that if $f$ is $C^1$, and there are no critical points
in $f^{-1}([a, b])$ (that we assume to be compact), then $M_b$ is diffeomorphic to $M_a$
and $M_a$ is a deformation retract of $M_b$. We extend this to locally Lipschitz functions,
following \cite{AgrachevContSelec}.
Note that in this setting $x$ is a critical point of $f$ iff $0 \in \partial_x f \subset T_x^*M$.

\begin{prop}\label{firstmorselem}
    Let $a < b \in \mathbb{R}$ such that $f^{-1}([a, b])$ is compact and free of critical points.
    Then there exists a locally Lipschitz homotopy $H : M_b \times [0, 1] \rightarrow M_b$ such
    that for all $s \in [0, 1]$, $H(\cdot, s)$ is a homeomorphism verifying
    \[ H(M_b \times \{s\}) \subset M_{b(1-s) + as} \]
\end{prop}

\begin{proof}
    We'll construct $V \in \operatorname*{Vec}(M)$ a smooth complete vector field on $M$
    such that, for all $x \in M$ such that $d_x f$ is defined :
    \begin{equation}\label{eq:Vsubgrad}
        d_x f (V(x)) \leq 0 \text{ and } d_x f(V(x)) \leq -1 \text{ if } x \in f^{-1}([a, b])
    \end{equation}
    Let $x \in f^{-1}([a, b])$. $x$ being regular and $\partial_x f$ convex, we find an affine hyperplane
    in $T^*_x M$ separating it from $0$. This gives us $v_x \in T_x M$ such that
    $\langle \partial_x f, v_x \rangle \leq -2$. By semi continuity, there exists
    an open neighbourhood $U_x$ of $x$ such that for all $y \in U_x$, $\langle \partial_y f, v_x \rangle \leq -1$.
    We choose this neighbourhood such that it has compact closure.
    \medskip \newline
    These $U_x$ cover $f^{-1}([a, b])$. Let $U_{x_1}, ..., U_{x_r}$ be a finite subcover, and let
    $\rho_1, ..., \rho_r$ be a partition of unity subordinated to this covering. We set
    \[\forall x \in M, V(x) = \sum_{i=1}^r \rho_i (x) v_{x_i} \]
    which indeed verifies \eqref{eq:Vsubgrad} : if $d_x f$ is defined, it is contained in $\partial_x f$,
    and $V$ is of compact support so it is complete.
    \medskip \newline
    Now observe that for $x \in f^{-1}([a, b])$, $t \mapsto f(e^{tV}x)$ is strictly decreasing.
    Denote by $t(x, s)$, $s \in [0, 1]$, the unique time such that $f(e^{t(x, s)V}x) = as + f(x)(1-s)$.
    The equation defining $t(x, s)$ is locally Lipschitz in $s, x$ and $t$, so by the Lipschitz implicit
    function theorem, $t$ is a locally Lipschitz function of $(x,s)$.
    \medskip \newline
    We can similarly define $t'$ a locally Lipschitz function of $x$ such that
    $f(e^{t'(x)V}x) = b$ for $x$ in the orbit of some $y \in f^{-1}(b)$ by $V$. Denote
    $O$ the set of $x \in M$ that are in such orbits.
    The homotopy is then given by
    \[ H : \begin{array}{cl}
        M_b \times [0, 1] &\longrightarrow M_b\\
        (x, s) &\longmapsto \left\{ \begin{array}{cl}
                e^{t(e^{t'(x)V}x, s)V}x & \text{ if } x \in O\\
                x &\text{ else}
            \end{array} \right.
    \end{array} \]
\end{proof}

\section{Morse theory of continuous selections}\label{sec:ContSelec}

\subsection{Definitions}

\quad We now restrict ourselves to continuous functions that are piecewise smooth with
finitely many pieces. We define those as continuous selections.

\begin{defi}
    A continuous function $f : M \rightarrow \mathbb{R}$ is a continuous selection if there
    exists $\varphi_1, ..., \varphi_n$ "regular" functions such that
    \[ \forall x \in M, f(x) \in \{ \varphi_1(x), ..., \varphi_n(x) \} \]
\end{defi}

The term "regular" is used loosely here to indicate that the $\varphi_i$ satisfy some
extra condition on top of being continuous. For what follows, we'll restrict ourselves
to continuous selections of smooth functions (but $C^2$ will suffice).
\medskip \newline
Note that a continous selection of locally Lipschitz functions is itself locally Lipschitz :
if $M$ is an open set of $\mathbb{R}^m$, and we assume that the functions $\varphi_i$ are
Lipschitz with constants $c_i$, then $f$ is Lipschitz with constant $\displaystyle \max_i c_i$.
Because smooth functions are locally Lipschitz, the functions we'll study will be too.
\medskip \newline
We can then relate $\partial f$ to the differentials of the $\varphi_i$. Indeed, a limit
of points of the form $d_x \varphi_i$ in $T^*M$ is of the same form. Define
the set of active indices at $x$,
\[ I(x) = \{ i \in \llbracket 1, n \rrbracket \mid \varphi_i(x) = f(x)\} \]
$\partial_x f$ is then the convex hull of some $d_x \varphi_i$, $i \in I(x)$.
It is possible that not all $d_x \varphi_i$ are used, if for example one of the functions isn't active
in a neighbourhood of $x$.
\bigskip

\quad We would like to define what it means for such a continuous selection to be Morse.
Let $f$ be a continuous selection of $\varphi_1, ..., \varphi_n$.
Following \cite{AgrachevContSelec}, we give three conditions for a function to be considered
Morse.
\begin{equation}\label{eq:morseCS1}
    \underline{\text{The graphs of the } \varphi_i, i \in \llbracket 1, n \rrbracket, \text{ are transversal}} \tag{C1}
\end{equation}
This condition has some important consequences :
for a non empty $I \subset \llbracket 1, n \rrbracket$, we define
\[ W_I := \{ x \in M \mid I(x) = I \} \]
Condition \eqref{eq:morseCS1} implies that $W_I$ is a submanifold of $M$ of codimension
$|I|-1$ : the intersection of the graphs of $\varphi_i, i \in I$ is a submanifold of
$M \times \mathbb{R}$ of codimension $|I|$ and $W_I$ is the pre-image of this intersection
through the diffeomorphism $x \mapsto (x, \varphi_i(x))$, $i \in I$, minus the closed set
$\bigcup_{J \supset I} W_J$.
\medskip \newline
This transversality condition also implies that for $x \in M$,
the differentials $d_x \varphi_i$ are affinely independant where $i \in I(x)$. By affinely
independant, we mean that one of the two equivalent conditions are satisfied :
for some/any $j \in I(x)$, the family $(d_x \varphi_i - d_x \varphi_j)_{\substack{i \in I(x) \\ j \neq i}}$
is linearly independant, or equivalently, the family
\[ \begin{pmatrix}
    d_x \varphi_i\\
    1
\end{pmatrix}, i \in I(x) \]
is linearly independant. This implies that $\partial_x f$ is a simplex with less than
$|I(x)| - 1$ vertices.
\medskip \newline
Note that for all $I \subset \llbracket 1, n \rrbracket$, $f_{\mid W_I}$ is a smooth function.
This motivates the second condition
\begin{equation}\label{eq:morseCS2}
    \underline{\text{For all non empty } I \subset \llbracket 1, n \rrbracket, f_{\mid W_I} \text{ is Morse}} \tag{C2}
\end{equation}
Consider $x \in M$ a critical point of $f$. Observe that $x$ must also be a critical point
of $f_{\mid W_{I(x)}}$. So
\[ \displaystyle T_x W_{I(x)} \subset \bigcap_{i \in I(x)} d_x \varphi_i \]
but the converse can fail if the $d_x \varphi_i$ are not linearly independant enough.
If this intersection is indeed larger, the geometry of the graph of $f$ does not correspond to what we
intuitively picture as a "non degenerate critical point" (see \ref{fig:morseCS3Illustr} for a 1
dimensional example). We then impose the following condition, ensuring equality.
\begin{equation}\label{eq:morseCS3}
    \underline{\forall x \in M, \forall j \in I(x), (d_x \varphi_i)_{\substack{i \in I(x) \\ i \neq j}} \text{ are linearly independant}} \tag{C3}
\end{equation}
Geometrically, this condition corresponds to requiring that $0$ is either outside or in the
interior of the simplex $\operatorname*{conv}\{ d_x \varphi_i \mid i \in I(x) \}$, but not on one of its faces.
Note that this condition can be relaxed by restricting it to the subset of $I(x)$ participating
in $\partial_x f$.
\bigskip

\quad A continuous selection $f$ is then called a Morse function if it verifies all three conditions.
We can also impose that these conditions only hold at critical points, and we get the same
theory.
\medskip \newline
Note that as in the smooth case, a generic continuous selection is Morse, i.e the set of
all $n$-tuples $(\varphi_1, ..., \varphi_n)$ such that these conditions hold at every critical
point is open dense in $C^2$ topology \cite{genericMorseCS}. We can see it intuitively by noticing that
\eqref{eq:morseCS1}
is a generic condition, any two functions can be slightly perturbed in order for their graphs
to be transversal, and any small perturbation of transversal graphs preserves transversality.
By genericity of smooth Morse functions, \eqref{eq:morseCS2} is also a generic condition. With
the geometric interpretation, we can also see that condition \eqref{eq:morseCS3} is generic.

\subsection{Normal form of a Morse continuous selection}

\quad We show here an equivalent of the second Morse lemma for the case of Morse continuous
selections. Noticing that $x \mapsto I(x)$ is upper semi-continuous and using the definitions
yields the following result.

\begin{lem}
    If $f : M \rightarrow \mathbb{R}$ is a Morse continuous selection, all its critical points
    are isolated.
\end{lem}

We'll then consider, up to restricting ourselves to a small enough chart,
that $f : \mathbb{R}^m \rightarrow \mathbb{R}$ is Morse and has no critical points other than $0$. We'll
write for shorthand $W := W_{I(0)}$, $k := |I(0)|-1$, $c := f(0)$, and assume that $I(0) = \llbracket 1, k+1 \rrbracket$.
Semi-continuity of the active index also allows us to assume that for $x \in \mathbb{R}^m$,
$I(x) \subset \llbracket 1, k+1 \rrbracket$.
\medskip \newline
Notice that because of condition \eqref{eq:morseCS1} and semi-continuity of the active
index, the family $(W_I)_{I \subset \llbracket 1, k+1 \rrbracket}$ gives a Whitney stratification
of $\mathbb{R}^m$. This observation motivates the following lemma, in the spirit of
Thom isotopy theory.

\begin{lem}\label{lem:prepNormForm}
    Let $h : \mathbb{R}^k \times \mathbb{R} \rightarrow \mathbb{R}$ be a continuous selection
    of at most $k+1$ smooth functions verifying condition \eqref{eq:morseCS1}.
    There exists a neighbourhood $U$ of $0$ and $\varepsilon > 0$, a vector field
    $X : U \times ]-\varepsilon, \varepsilon[ \rightarrow \mathbb{R}^k \times \mathbb{R}$ verifying
    \begin{enumerate}
        \item For $x \in U, t \in ]-\varepsilon, \varepsilon[$ we have
        \[ X(x, t) = (\tilde{X}(x, t), 1) \]
        \item If $(x,t) \in W_I$ for some active index set $I$, then $X$
        is tangent to $W_I$ at all times and the restriction to $W_I$ is $C^1$
        \item $(x, t) \mapsto h(x, t)$ is constant along integral curves of $X$
        \item The flow of $X$ is continuous.
    \end{enumerate}
\end{lem}

The use of this lemma requires a parameter dependent continuous selection. For the proof of
the main theorem, we will have to build one with some boundary conditions wich is done
in the following way

\begin{lem}\label{lem:extendingTimeCS}
    Let $\varphi_1, ..., \varphi_{k+1} : \mathbb{R}^k \times \mathbb{R} \rightarrow \mathbb{R}$
    be smooth functions verifying conditions \eqref{eq:morseCS1} and of value $0$ on $\{0\}^k \times \mathbb{R}$.
    Let $h^0 : \mathbb{R}^k \rightarrow \mathbb{R}$ be a continuous selection of
    $\varphi_1(\cdot, 0), ..., \varphi_{k+1}(\cdot, 0)$. Then there exist a neighbourhood $U$
    of $0$ and
    a continuous selection $h : U \times [0, 1] \rightarrow \mathbb{R}$ of
    $\varphi_1, ..., \varphi_{k+1}$ such that $h(\cdot, 0) = h^0$.
\end{lem}

\begin{proof}
    For all $i < k+1$ define $g_i(x, t) = \varphi_i(x, t) - \varphi_{k+1}(x,t)$. By
    affine independance the family $(d_0 g_i)_i$ is linearly independant. Using the
    implicit function theorem, there are coordinates on a neighbourhood $U \times J$
    of $0$ linearizing all $g_i$, while sending $\mathbb{R}^k \times \{0\}$ to a hyperplane.
    We assume without loss of generality that this hyperplane is $\mathbb{R}^k \times \{0\}$.
    It follow that for every $i, j$, $\varphi_i - \varphi_j$ is a linear function in these
    new coordinates.
    \medskip \newline
    Denote $W_I \subset \mathbb{R}^k \times \{0\}$ the set of points $(x, 0)$ such that the active
    index of $h^0$ at $x$ is equal to $I \subset \llbracket 1, k+1 \rrbracket$. We also define
    $L = \{ g_i = 0 \mid i \in \llbracket 1, k+1 \rrbracket \}$ which is a linear subspace of
    dimension $1$. If we now define for all $I$, $C_I = W_I + L$, it defines a partition
    of $\mathbb{R}^{k+1}$.
    \medskip \newline
    Observe that for $z = (x, 0) + u$, $(x, 0) \in W_I, u \in L$ we have for $i, j \in I$ that
    \[ \varphi_i(z) - \varphi_j(z) = \varphi_i(x, 0) - \varphi_j(x, 0) = 0 \]
    by linearity. We can then define for all $(x, t) \in C_I$ the function $h(x, t) = \varphi_i(x, t)$
    for any $i \in I$. This $h$ defines a continuous selection coinciding with $h^0$ on
    $\mathbb{R}^k \times \{0\}$ in the new coordinates.
    \medskip \newline
    We now want to extend this selection on the interval $[0, 1]$. By repeating the
    argument for some starting time different than $0$, connectedness and
    compactness yield that there is a neighbourhood $U$ of $0$ such that we can define $h$ on $U \times [0, 1]$.
\end{proof}

We are now ready to prove the main result of this section.

\begin{theorem}[Normal form]
    If $0$ is a critical point of $f$, then there exists continuous local coordinates around $0$,
    $\psi : \mathbb{R}^k \times \mathbb{R}^{m-k} \rightarrow U$ such that
    \[ \forall y \in \mathbb{R}^k, \forall z \in \mathbb{R}^{m-k}, f(\psi(y,z)) = \lambda(y) + q(z)\]
    where $\lambda$ is a continuous selection of $k+1$ affinely independant linear functions, and $q$ is of the form
    \[ \forall z \in \mathbb{R}^{m-k}, q(z) = c -\sum_{i=1}^{\mu} z_i^2 + \sum_{i=\mu+1}^{m-k} z_i^2 \]
    with $0 \leq \mu \leq m-k$.
\end{theorem}

\begin{proof}
    We first restrict ourselves to $O$ a neighbourhood of $0$ such that $W$ can be considered
    rectified in some smooth coordinates i.e, $W \cap O = 0 \times \mathbb{R}^{m-k}$. By \eqref{eq:morseCS2}
    we can also use the smooth Morse lemma to get $f(0, z) = q(z)$. For $(y, z) \in \mathbb{R}^m$,
    write $f^0(y, z) = f(y, z) - f(0, z)$. We will build
    a locally invertible map $\psi : \mathbb{R}^m \rightarrow \mathbb{R}^m$ such that
    $\psi(0) = 0$ and for $(y,z)$ close enough to $0$, $f^0 \circ \psi (y, z)$ is a continuous selection of
    linear maps independant of $z$.
    \bigskip \newline
    Assume that all $\varphi_i$ are of value $0$ at $0$, up to replacing $f$ by $f^0$.
    We will show that up to coordinate change, $f^0$ is only a function of $y$ and not of $z$.
    This can be done in the following way : first, by condition \eqref{eq:morseCS3}, we can
    assume that the $k$ first $\varphi_i$ are linearized, i.e equal to $dy_i$ in
    a neighbourhood of $0$. The last $\varphi_{k+1}$ could still depend on $z$.
    \medskip \newline
    To fix this we apply lemma \ref{lem:prepNormForm} to
    $f^0 : \mathbb{R}^{m+k-1} \times \mathbb{R} \rightarrow \mathbb{R}$.
    Denote $\Phi(s, y, z)$ the point obtained by integrating $X$ for time $s$ starting at
    $(y, z)$. The following local homeomorphism
    \begin{equation}\label{eq:constrPhiNormform}
        \phi(y, z) = \Phi(-z_{m-k}, y, z) + (0, ..., 0, z_{m-k})
    \end{equation}
    verifies $f^0 \circ \phi^{-1}(y, z) = f^0(y, z_1, ..., z_{m-k-1}, 0)$. By repeating the process
    we eliminiate the dependance in $z$.
    \bigskip \newline
    Note that in the new coordinates, $\varphi_i = dy_i$ for $i < k + 1$. To write $f^0$ as
    a continuous selection of linear functions, it suffices to linearize $\varphi_{k+1}$.
    For that, for all $t \in [0, 1]$ define
    \[ \varphi_{k+1}^t = (1-t)\varphi_{k+1} + t d_0 \varphi_{k+1} \]
    Note that for $y$ close enough to $0$ and any $t\in [0, 1]$, $d_y \varphi_{k+1}^t$ is close
    to $d_y \varphi_{k+1}$, hence the functions $dy_1, ..., dy_k, \varphi_{k+1}^t$ verify
    \eqref{eq:morseCS1}.
    By lemma \ref{lem:extendingTimeCS}, we define $h(y,t)$ for all $y$ close enough to $0$ and $t \in [0, 1]$ as
    a continuous selection of $dy_1, ..., dy_k, \varphi_{k+1}^t$ such that
    it equals $f^0$ at $t=0$.
    \medskip \newline
    We then apply lemma \ref{lem:prepNormForm}
    and get $U$ an even smaller neighbourhood and $X$ a vector field on a neighbourhood
    of $(0, 0)$. Up to shrinking $U$, we can assume that for all $t^*$ there is a
    time $s(t^*) > 0$ such that for all $y \in U$, we can integrate $(y, t^*)$ for time
    $s, |s| < s(t^*)$. This gives a covering of $[0, 1]$ by intervals of the form
    $]t^*-s(t^*), t^*+s(t^*)[$, so we can still cover it with finitely many of these intervals.
    \medskip \newline
    Assume for simplicity 
    that we can flow for time $1$ from all $(y, 0)$, $y \in U$ i.e $[0, 1]$ can be covered with
    only one such interval. We construct a local 
    homeomorphism $\phi$ similarly to \eqref{eq:constrPhiNormform} such that 
    \[ f^0(y) = h(y, 0) = h \circ \phi^{-1} (y, 1) \]
    so $f^0$ can be written in some coordinates as a continuous selection of linear functions.
    If we need more than one interval to cover $[0, 1]$ we can decompose this into finitely many
    homeomorphisms going from $0$ to $1$ in successive steps.
    \bigskip \newline
    By composing all the steps, we indeed constructed the local homeomorphism $\psi$.

\end{proof}

\quad We now try to understand better the structure of the non smooth component that appears in
the normal form theorem. It turns out that it can be encoded in a purely
combinatorial way, as the following theorem shows \cite{PLminmax}.

\begin{theorem}\label{th:minmax}
    Let $\lambda : \mathbb{R}^k \rightarrow \mathbb{R}$ be a continuous selection of
    $k+1$ affinely independant linear functions, $\lambda_1, ..., \lambda_{k+1}$. Then there exists a unique family
    $S_1, ..., S_l$ of subsets of $\llbracket 1, k+1 \rrbracket$ verifying that
    \begin{itemize}
        \item $S_i \not \subset S_j$, $i \neq j$
        \item $\lambda$ can be written as
        \[ \lambda(y) = \min_{i} \max_{j \in S_i} \lambda_j(y) \]
    \end{itemize}
\end{theorem}

\begin{proof}
    Let's first tackle uniqueness. Let $S_1, ..., S_l$, $D_1, ..., D_r$ be two such distinct families.
    Assume without loss of generality that $S_1 \neq D_j$. We can also assume that for all
    $1 \leq j \leq r$, $S_1 \not\supset D_j$. If it is the case, then for all $1 < i \leq l$
    we have $D_j \not\supset S_i$ by assumption on $S$. We can then swap $S$ and $D$ to get
    what we want.\newline
    By affine independance, the following linear system has a solution
    \[ \lambda_i(y) + \mu = c_i, i \in \llbracket 1, k+1 \rrbracket \]
    where the unknowns are $y, \mu$ and $c_i = 0$ if $i \in S_1$ and $1$ otherwise.
    It follows that, if we use the representation with $D_i$, we get $\lambda(y) = 1 - \mu$
    and if we use the one with the $S_i$, $\lambda(y) = -\mu$.
    \medskip \newline
    Now for existence. For all permutation $\nu \in \mathfrak{S}_{k+1}$ define the cone
    \[ K_{\nu} = \{ y \in \mathbb{R}^k \mid \lambda_{\nu(1)}(y) \leq ... \leq \lambda_{\nu(k+1)}(y) \} \]
    and the function $s : \nu \in \mathfrak{S}_{k+1} \mapsto i \in \llbracket 1, k+1 \rrbracket$ where $i$
    is such that $\lambda_{\nu(i)} = \lambda$ on $K_{\nu}$. If we assume that for all $y$ and $\nu$ we have
    \[ \lambda(y) \leq \max \{ \lambda_{\nu(1)}(y), ..., \lambda_{\nu(s(\nu))}(y) \} \]
    then it suffices to consider the family of sets $S_{\nu} = \{\nu(1), ..., \nu(s(\nu)) \}$
    and simplify these by removing those who are contained in one another.
    \medskip \newline
    We now give a broad idea of the proof of the claim. The result is true by definition
    on $K_{\nu}$. For any $y \in \mathbb{R}^k$, set $y' \in K_{\nu}$ and consider the evolution
    of the order of $\lambda_i$ along the line between $y$ and $y'$ : the value of each $\lambda_i$
    evolves at a constant rate. Whenever the current $\lambda_i$ defining $\lambda$ encounters
    another $\lambda_j$ and $\lambda$ becomes defined as $\lambda_j$ afterwards, either
    $\lambda_j$ is smaller than $\lambda_i$ after the encounter, or the opposite. If we denote
    $I$ the set of indices such that the functions are smaller than $\lambda_j$ before the
    encounter, then in either case we see that $\lambda$ stays smaller than the maximum
    of $\lambda_{i'}$ $i' \in I$. This concludes the proof because the line starts at $K_{\nu}$.
\end{proof}

\subsection{Describing the homology of sublevel sets}

\quad Our goal for this section is to describe more precisely the transition that happens
as the critical point of $f$ is passed, where $f$ is still a Morse continuous selection
defined on $\mathbb{R}^m$ with $0$ as its only critical point.
\medskip \newline
The normal form of $f$ and theorem \ref{th:minmax} allows us to fully describe the behaviour
of $f$ with purely combinatorial information, a smooth Morse index and a more complicated set
representation. We can encode the combinatorial structure that appears in its non smooth part in a way
that will give us topological and homotopical informations about the sublevel sets : define the
simplical complex $K(S_1^c, ..., S_l^c)$ on $k+1$ vertices whose faces are given by the complements in
$\llbracket 1, k+1 \rrbracket$ of the sets $S_i$ describing the non smooth part of $f$.
\medskip \newline
Another thing to note is that when put in normal form, $f$ is a quasi homogeneous function, i.e
there exists some real numbers $\alpha_1, ..., \alpha_m > 0$ such that for all $t > 0, x \in \mathbb{R}^m$,
we have
\[ f(t^{\alpha_1} x_1, ..., t^{\alpha_m} x_m) = tf(x) \]
In this case we have $f(ty, t^{1/2}z) = tf(y,z)$ if we assume that $f(0) = 0$, assumption we'll
make without loss of generality. For a quasi homogenous function $h$, we'll denote $\delta_t$
the power-scaling map such that $h(\delta_t x) = th(x)$. The following lemmas on quasi
homogeneous functions will prove useful for describing the homotopy type of sublevel sets.

\begin{lem}\label{lem:pairconehomequiv}
    Let $h : \mathbb{R}^m \rightarrow \mathbb{R}$ be a quasi homogeneous locally Lipschitz
    function having only $0$ as a critical point. We have the following homotopy equivalence
    \[ (M_1, M_{-1}) \simeq (C_{h^{-1}(-1)}, h^{-1}(-1)) \]
    where $M_1, M_{-1}$ are the sublevel sets and $C_{h^{-1}(-1)}$ is the cone over $h^{-1}(-1)$.
\end{lem}

\begin{proof}
    Let $\mathbb{S}$ denote the unit sphere. Define a smooth $V$ over $\mathbb{S}$ such that
    for all $x \in \mathbb{S}$, $V(x) \in \mathbb{R}^m$ and $\langle \partial_x h, V(x) \rangle \leq -1$.
    Extend $V$ into a vector field on $\mathbb{R}$ such that it commutes with $\delta_t$ for
    all $t > 0$, i.e for $x \in \mathbb{S}$, $V(\delta_t x) = d_x \delta_t( V(x) )$. It
    follows that $s \mapsto h(e^{sV} x)$ is strictly decreasing for all $x \neq 0$.
    \medskip \newline
    From this observation, we define similarly to proposition \ref{firstmorselem}, for all
    $x \in h^{-1}(]-\infty, 0[)$, $p(x)$, the only point of the orbit of $x$ in $h^{-1}(-1)$.
    We also define for all $x \in h^{-1}(-1), s \in [0, 1[$, $t(x,s)$ such that it verifies
    $h(e^{-t(x,s)V}x) = h(x) + s$. If we then denote by
    $\pi : [0,1] \times h^{-1}(-1) \rightarrow C_{h^{-1}(-1)}$ the projection map, where the
    top of the cone is the point at height $s = 1$, the homotopy equivalence maps are the
    following
    \[ F : \begin{array}{cl}
        M_1 &\longrightarrow C_{h^{-1}(-1)}\\
        x &\longmapsto \left\{ \begin{array}{cl}
            \pi(0, p(x)) & x \in M_{-1}\\
            \pi(h(x)+1, p(x)) & x \in h^{-1}([-1, 0])\\
            \pi(1, 0) & x \in h^{-1}([0, 1]) 
        \end{array} \right.
    \end{array}
    \]
    \vspace*{0.5cm}
    \[ G : \begin{array}{cl}
        C_{h^{-1}(-1)} &\longrightarrow M_1\\
        \pi(s, x) &\longmapsto e^{-t(x,s)}x
    \end{array}
    \]
    We can see that $F \circ G = \operatorname{id}_{C_{h^{-1}(-1)}}$. $G \circ F$ is homotopic
    to the identity : we can use $V$ to contract $M_{-1}$ onto $h^{-1}(-1)$, and also to
    contract $h^{-1}([0, 1])$ onto $h^{-1}(0)$.
\end{proof}

\quad If $X$ and $Y$ are two topological spaces, we define the join of $X$ and $Y$,
denoted by $X * Y$, as the quotient space of $X \times Y \times [0, 1]$ by the following
equivalence relation
\[ \forall x, x' \in X, \forall y, y' \in Y, (x,y,0) \sim (x, y', 0) \text{ and } (x,y,1) \sim (x',y,1)\]
Intuitively, we join together the spaces $X$ and $Y$ with all possible segments between the two.
Some examples of join include $X * \{y\}$ wich is the cone of $X$,
$X * \mathbb{S}^0$ which is the suspension of $X$. More generally the join of $X$ with $\mathbb{S}^n$
is an $n+1$-fold suspension. 
\medskip \newline
By using the same ideas as in lemma \ref{lem:pairconehomequiv}, one can prove the following
useful lemma.

\begin{lem}
    Let $h : \mathbb{R}^k \times \mathbb{R}^{m-k} \rightarrow \mathbb{R}$ be a quasi homogeneous function such that
    for all $(y, z) \in \mathbb{R}^m$, we have $h(y, z) = a(y) + b(z)$ where $a$ and $b$
    are quasi-homogeneous locally Lipschitz with respective exponents $\alpha, \beta > 0$. We have the following
    homotopy equivalence
    \[ h^{-1}(-1) \simeq a^{-1}(-1) * b^{-1}(-1) \]
\end{lem}

\quad If we write $f(y, z) = \lambda(y) + q(z)$ as described by the normal form theorem,
what remains to be done to understand the relative homology of $(M_1, M_{-1})$ is to
describe $q^{-1}(-1)$ and $\lambda^{-1}(-1)$. The first one has the homotopy type of
$\mathbb{S}^{\mu}$ where $\mu$ is the smooth Morse index of $f$, by classical Morse theory.
\medskip \newline
For the second, note that we have the following homotopy equivalence
$\lambda^{-1}(-1) \simeq \lambda^{-1}(]-\infty, 0[)$. Using the min max form, we get
\[ \lambda^{-1}(]-\infty, 0[) = \bigcup_i \bigcap_{j \in S_i} \lambda_j(]-\infty, 0[) \]
Note that each intersection is a convex cone we'll denote by $C_i$. We'll
relate this to the simplical complex $K(S_1^c, ..., S_l^c)$ by using the nerve of the
family $C = (C_i)_i$, i.e we'll look at the following set
\[ N(C) := \{ J \subset \llbracket 1, l \rrbracket \mid \bigcap_{j \in J} C_j \neq \varnothing\} \]
From the next lemma, it follows that the nerve of $N(C)$ is exactly the one of the family of
sets $S_1^c, ..., S_l^c$.

\begin{lem}
    Let $J \subset \llbracket 1, k+1 \rrbracket$. $J$ is the whole set if and only if
    \[ \bigcap_{j \in J} \lambda_j^{-1}(]-\infty, 0[) = \varnothing \]
\end{lem}

\begin{proof}
    We first show that if $J = \llbracket 1, k+1 \rrbracket$, we indeed get an empty intersection.
    Recall that $0$ is a critical point of $f$, so it is a critical point of $\lambda$. We
    then have
    \[ 0 \in \partial_0 \lambda \subset \operatorname{conv}\{ \lambda_1, ..., \lambda_{k+1} \} \]
    So there exists $\mu_1, ..., \mu_{k+1}$ with sum $1$ such that $0 = \sum_i \mu_i \lambda_i$.
    We then see that all $\lambda_i$ cannot be strictly negative at the same time.
    \medskip \newline
    We now show that if $J \neq \llbracket 1, k+1 \rrbracket$, the intersection is not empty.
    Condition \eqref{eq:morseCS3} gives us linear independance of the family $(\lambda_j)_{j \in J}$.
    The linear system $\lambda_j(y) = -1, j \in J$ then has a solution, where the unknown is $y$.
\end{proof}

Because every $C_i$ is convex, their intersections are contractible. Applying Leray's nerve
theorem, $\lambda^{-1}(-1)$ has the homotopy type of the simplical complex given
by the nerve. By virtue of what we discussed, this complex is exactly $K(S_1^c, ..., S_l^c)$.
The homotopy type of $f^{-1}(-1)$ is then given by
\[ f^{-1}(-1) \simeq K(S_1^c, ..., S_l^c) * \mathbb{S}^{\mu} \]

By recalling that a join with a sphere is a suspension, we get the following result in homology.

\begin{theorem}
    For $i \geq 0$, we have
    \[ H_{i+\mu}(M_1, M_{-1}) = \tilde{H}_{i-1}(K(S_1^c, ..., S_k^c)) \]
\end{theorem}

\end{flushleft}

\section{First properties of the euclidean distance}

\quad In this section, we will work in the euclidean space $\mathbb{R}^m$. Some of what
we'll do here can be generalized to a more general Riemannian manifold. All throughout,
$X \subset \mathbb{R}^m$ will be a smooth manifold unless specified otherwise,
$Y \subset \mathbb{R}^m$ a closed set.
\medskip \newline
The main function we will study is the euclidean distance from $Y$ restricted to $X$, i.e
\[ \forall x \in X, \operatorname{dist}_Y(x) = \min_{y \in Y} |x - y| \]
where $|\cdot|$ is the euclidean norm. This function is locally Lipschitz. In what will
follow, we identify $(\mathbb{R}^m)^*$ with $\mathbb{R}^m$ via the euclidean structure.
$\partial_x ({\operatorname{dist}_Y}_{\mid X})$ will sometimes be identified with a subset of $T_x X$,
where $x \in X$.
\medskip \newline
Our main objectives will be to understand the subdifferential of this function and to find
a setting where Morse theory of continuous selections can be applied.

\subsection{The subdifferential of the distance function}

\quad Our goal in this section will be to compute explicitly the subdifferential of
${\operatorname{dist}_Y}_{\mid X}$, which gives rise to a very nice and intuitive geometric
picture. We first require some results on "infinite
continuous selections", which is indeed how $\operatorname{dist}_Y$ is defined.
\medskip \newline
Let $\varphi : \mathbb{R}^m \times Z \rightarrow \mathbb{R}$ be a continuous function,
where $Z$ is a compact space. We can define a family $(\varphi_z)_{z \in Z}$ of continuous
functions, where $\varphi_z = \varphi(\cdot, z)$. Similar to the finite case we can define
a "compact continuous selection" as a continuous $f$ such that for all $x \in \mathbb{R}^m$
\[ f(x) \in \{ \varphi_z(x) \mid z \in Z \} \]
Similarly to the finite case, we define the set of active indices at $x$, $Z(x)$ as the set
of all $z$ that take the value of $f$ at $x$. The following result is a useful generalization of
a property of finite continuous selections.

\begin{lem}
    The function $x \mapsto Z(x)$ is upper semi-continuous.
\end{lem}

\begin{proof}
    Let $x \in \mathbb{R}^m$ and let $V$ be an open neighbourhood of $Z(x)$.
    Consider for all $n \in \mathbb{N}$ the closed ball $B_n$ of center $x$ and radius
    $2^{-n}$. Set
    \[ A_n = \{ z \in Z \mid \exists y \in B_n, \varphi_z(y) = f(y)\} \]
    $A_n$ is closed : indeed if we write $g := \varphi - f$, we can write $A_n$ as
    $\operatorname{proj}_Z(g^{-1}(0) \cap (B_n \times Z))$, and $g^{-1}(0) \cap (B_n \times Z)$ is compact
    so we get what we want.
    \medskip \newline
    Note that the intersection $\bigcap_n (A_n \backslash V)$ is empty : if there was a point $z$
    in such intersection, there would be a sequence $x_k \rightarrow x$ along which $\varphi_z(x_k) = f(x_k)$,
    so $z \in Z(x) \subset V$. By compactness, and because $(A_n)$ is decreasing, there is
    some $n$ such that $A_n \subset V$. Let $y \in B_n$ for such $n$. We have
    \[ Z(y) \subset A_n \subset V \]
    so we get the desired result.
\end{proof}

The following theorem, which is a slightly modified version of Danskin's theorem,
will allow us to compute explicitly subdifferentials in a particular case.

\begin{theorem}[Danskin's \cite{DanskinsProof}]\label{thm:DanskinPlus}
    Assume that for all $z$, $\varphi_z$ is $C^1$, $z \mapsto d_x \varphi_z$ is
    continuous and $f = \max_{z \in Z} \varphi_z$.
    \begin{enumerate}
        \item $f$ is convex and for all $x, v \in \mathbb{R}^m$
        \[ D_x f(v) = \max_{z \in Z(x)} d_x \varphi_z(v) \]

        \item $f$ is locally Lipschitz and we have
        \[ \partial_x f = \operatorname{conv}\{ d_x \varphi_z(v) \mid z \in Z(x) \} \]
    \end{enumerate}

\end{theorem}

\begin{proof}
    We'll need 1. for 2.
    \begin{enumerate}
    \item The convexity of $f$ is a classical result from convex analysis. Notice that
    for $t > 0$
    \[ \frac{f(x + tv) - f(x)}{t} \geq \frac{\varphi_z(x+tv) - \varphi_z(x)}{t} \]
    for $z \in Z(x)$, so we get $D_x f(v) \geq d_x \varphi_z(v)$. For the other inequality,
    consider a positive sequence $t_n \rightarrow 0$. For all $n$, select $z_n \in Z(x+t_nv)$,
    by compactness assume it converges to $z^* \in Z$. By maximality of $f$, we have
    $\varphi_{z^*}(x) \geq \varphi_z(x), z \in Z$, so $z^* \in Z(x)$. By convexity of $f$
    and $\varphi_{z_n}$, we get
    \begin{align*}
        D_x f(v) &\leq \frac{f(x+t_n v) - f(x)}{t_n}\\
        &= \frac{\varphi_{z_n}(x+t_n v) - \varphi_z^*(x)}{t_n}\\
        &\leq \frac{\varphi_{z_n}(x+t_n v) - \varphi_{z_n}((x+ t_n v) - t_n v)}{t_n}\\
        &\leq d_x \varphi_{z_n}(v)\\
        &\rightarrow d_x \varphi_{z^*}(v)
    \end{align*}
    so $D_x f(v)$ is indeed the maximum.
    
    \item All $\varphi_z$ are locally Lipschitz. By compactness of $Z$, we can find a Lipschitz
    constant for $f$.
    \medskip \newline
    We now show the equality involving $\partial f$. For that we use the results of the
    discussion in section \ref{sec:subdiffconvex}. For the right to left inclusion,
    it suffices to observe that in virtue of 1., for $z \in Z(x)$
    \[ \forall v \in \mathbb{R}^m, f(x+v) \geq f(x) + d_x \varphi_z(v) \]
    Assume that there is some $l \in \partial_x f$ not in the convex hull of $d_x \varphi_z$,
    $z \in Z(x)$. Because this convex hull is compact, we choose a strictly separating
    hyperplane : we choose $v$ and $c$ such that
    \[ \langle l, v \rangle > c > \langle d_x \varphi_z, v \rangle, z \in Z(x) \]
    This contradicts the fact that $D_x f (v) = \max_{h \in \partial_x f} \langle h, v \rangle$,
    because $D_x f(v) = \max_{z \in Z(x)} \langle d_x \varphi_z, v \rangle$
    \end{enumerate}
\end{proof}

\quad We are now ready to compute the subdifferential of ${\operatorname{dist}_Y}_{\mid X}$.

\begin{theorem}\label{thm:subdiffdist}
    Let $x \in X \backslash Y$. We have
    \[ \partial_x ({\operatorname{dist}_Y}_{\mid X}) = \operatorname{proj}_{T_x X} \left( \operatorname{conv}\left\{ \frac{x-y}{|x-y|}
        \mid y \in Y(x) \right\} \right)\]
    where $\operatorname{proj}_{T_x X}$ is the orthogonal projection onto $T_x X$.
\end{theorem}

\begin{proof}
    We first compute the subdifferential in the case where $X = \mathbb{R}^m$, and we
    deduce the general case.
    \medskip \newline
    Let $\eta_Y = \frac{1}{2} \operatorname{dist}^2_Y$. We write for $x \in \mathbb{R}^m \backslash Y$
    \[
        \eta_Y(x) = \frac{1}{2} |x|^2 - \left( \max_{y \in Y} \langle x, y \rangle - \frac{1}{2} |y|^2 \right)
    \]
    The function $\varphi_y(x) = \langle x, y \rangle - \frac{1}{2} |y|^2$ is convex. If we denote
    by $h(x)$ the term in parenthesis, theorem \ref{thm:DanskinPlus} yields
    \[ \partial_x h = \operatorname{conv}\{\nabla_x \varphi_y \mid y \in Y(x)\}
    = \operatorname{conv}(Y(x)) \]
    We can apply the theorem because in a small neighbourhood of $x$, the active indices stay
    confined to a compact set.\newline
    By using that $\eta_Y = \frac{1}{2} | \cdot | - h$, we get
    \[ \partial_x \eta_Y = \{x - y \mid y \in Y(x) \} \]
    If we now write $\operatorname{dist}_Y = \sqrt{2\eta_Y}$, and use the subdifferential chain
    rule applied to the diffeomorphism $\sqrt{\cdot}$ (because $\eta_Y(x) \neq 0$, $x \notin Y$) we get
    \begin{align*}
        \partial_x \operatorname{dist}_Y &= \frac{1}{\sqrt{2\eta_Y(x)}} \operatorname{conv}\{x-y \mid y \in Y(x) \}\\
        &= \operatorname{conv}\left\{ \frac{x-y}{|x-y|} \mid y \in Y(x) \right\}
    \end{align*}
    because $\eta_Y(x) = \frac{1}{2}|y-x|^2$ for all $y \in Y(x)$.
    \bigskip \newline
    Denote $g = \operatorname{dist}_Y$, and $f = {\operatorname{dist}_Y}_{\mid X}$.
    We will show that $\partial_x f = \operatorname{proj}_{T_x X} (\partial_x g)$.
    \medskip \newline
    We start from left to right.
    We want to show that $x \mapsto \operatorname{proj}_{T_x X} (\partial_x \operatorname{dist}_Y)$
    is upper-semi continuous (where the set is seen in $TX$), and contains $\nabla_x f$ whenever
    defined. The first part is easy by semi-continuity of the original function. For the
    second, we assume $\nabla_x f$ is defined for some $x \in X$. By the computations made
    earlier and Danskin's theorem, we write
    \[ D_x g (v) = \min_{y \in Y(x)} \langle \nabla_x g_y, v \rangle \]
    where $g_y(x) = |x - y|$. Note that because $\nabla_x f$ exists, this is linear if
    restricted to $T_x X$. By using linearity, we find that there is $y \in Y(x)$ such that for all
    $v \in T_x X$
    \[ D_x g (v) = \langle \nabla_x g_y, v \rangle = \langle \nabla_x f, v \rangle \]
    Hence $\nabla_x g_y$ projects onto $\nabla_x f$, so we have what we want. The inclusion
    then follows from minimality of subdifferentials.
    \medskip \newline
    For the inclusion from right to left, we consider $x_k \rightarrow x \in X$ a sequence
    such that $\nabla_{x_k} g$ is well defined and converges, and we will show that it
    is a subgradient of ${\operatorname{dist}_{Y}}_{\mid X}$ up to projection. We consider
    up to restricting to a neighbourhood of $x$, that $X = \mathbb{R}^{n}$ where $n = \dim(X)$.
    Let $(V_k)_k$ be a decrasing sequence of open neighbourhoods of $\partial_x f$ converging
    to it. By semi-continuity of $\partial g$ and up to extracting, $\nabla_{x_k} g \in V_k \times \mathbb{R}^{m-n}$,
    So the projection of $\nabla_{x_k} g$ converges to a point in $\partial_x f$, we got
    equality.
\end{proof}

\subsection{Special points of the distance function}

\quad We now consider $X \subset \mathbb{R}^m$ to be some non specific submanifold.
We denote in this section $g : (a, b) \mapsto |a - b|^2$ and $g_a : x \mapsto |a - x|^2$
for all $a \in \mathbb{R}^m$.\newline
Following \cite{AntonioMorseThEuclDist}, we define

\begin{defi}[Medial axis]
    We define $M_Y$, the medial axis of $Y$ as
    \[ M_Y = \{ x \in \mathbb{R}^m \mid |Y(x)| \geq 2 \} \]
\end{defi}

Some other special points of the distance function include the cut-locus of $Y$, that we'll
denote $\operatorname{CLoc}(Y)$ and the central set of $Y$ that we denote
$C(Y)$, introduced respectively by Thom \cite{ThomCutLocus} and Milman \cite{MilmanMedialCut}.
\begin{defi}[Central set]
    An open ball $B \subset \mathbb{R}^m \backslash Y$ will be called maximal if there is no
    open ball in $\mathbb{R}^m \backslash Y$ strictly containing $B$. The central set $C(Y)$ is
    then the set of the centers of all maximal balls.
\end{defi}

If $Y$ is smooth we can also define the following set

\begin{defi}[Cut-locus]
The cut-locus of $Y$ is defined as follows
\[ \operatorname{CLoc}(Y) = M_Y \cup \{ x \in \mathbb{R}^n \mid {g_x}_{\mid Y} \text{ has a degenerate global minimum} \} \]
\end{defi}

If we assume that $Y$ is a submanifold of $\mathbb{R}^m$, we can link these seemingly
differents concepts by the following proposition, generalizing the arguments from \cite{MilmanMedialCut}.

\begin{prop}\label{prop:equalitySpecialPoints}
    Assume $Y$ is a smooth submanifold of $\mathbb{R}^m$. Then $\overline{M_Y} = \operatorname{CLoc}(Y) = C(Y)$
\end{prop}

\begin{proof}
    We first show $\overline{M_Y} \subset \operatorname{CLoc}(Y)$. Consider a sequence
    $x_k \rightarrow x$, where $x_k \in M_Y$ and $x$ is not. Let $y$ be the single element in $Y(x)$,
    and for all $k$, $y_k, y'_k \in Y(x_k)$ two different minimizers.
    Then for all $k$, $\nabla_{y_k} g_{x_k} - \nabla_{y'_k} g_{x_k} = 0$
    and we can show that $\operatorname{Hess}_y ({g_x}_{\mid Y})$ vanishes in some direction $u \in T_y Y$, defined as
    an extracted limit of $\frac{y_k - y'_k}{|y_k - y'_k|}$. Hence $x \in \operatorname{CLoc}(Y)$.
    \bigskip \newline
    We now show $\operatorname{CLoc}(Y) \subset C(Y)$. We have $M_Y \subset C(Y)$ : indeed
    if for $x \in M_Y$ there is a ball $B'$ containing $B := B(x, \operatorname{dist}_Y(x))$,
    then $\partial B'$ and $\partial B$ must intersect tangentially at $Y(x)$. This set has
    at least two points so we find that the center of $B'$ must be $x$, so $x \in C(Y)$.
    \medskip \newline
    Let $x \in \operatorname{CLoc}(Y)$ such that $x \notin M_Y$. So ${g_x}_{\mid Y}$ has a degenerate
    global minimum. Let $y$ be this minimum, $R = |x-y|$ and denote
    \[ \nu = \frac{x-y}{R} \in N_y Y \]
    Minimality gives us $\operatorname{Hess}_y ({g_x}_{\mid Y}) \succeq 0$, hence if we denote by $\kappa_i$
    the principal curvatures in direction $\nu$ of $Y$ i.e the eigenvalues of
    the second fondamental form in direction $\nu$ at $y$,
    then $\kappa_i \leq \frac{1}{R}$. By \cite{centerOfCurvMilnor},
    $\operatorname{Hess}_y ({g_x}_{\mid Y})$ being degenerate implies that for some $i$,
    $\kappa_i = \frac{1}{R}$.
    \medskip \newline
    If we now assume that there is a strictly larger maximal ball $B'$ containing $B(x, R)$,
    $B'$ must also be tangent to $\partial B(x, R)$ at $y$. The center of $B'$ then lies
    on a line with $y$ and $x$. Denote $R' > R$ the radius of $B'$. Applying the same arguments
    to $B'$, $\kappa_j \leq \frac{1}{R'}$ for all $j$. But for some $i$, $\kappa_i = \frac{1}{R} > \frac{1}{R'}$,
    contradiction. So $B(x, R)$ is a maximal ball, $x \in C(Y)$.
    \bigskip \newline
    Lastly, we show that $C(Y) \subset \overline{M_Y}$. Assume for contradiction that there
    is some $x \in C(Y)$ not in $\overline{M_Y}$. We find $\varepsilon > 0$ such that for
    $z \in B := B(x, \varepsilon)$, $Y(z)$ is a singleton. We can define
    $\eta : B \rightarrow Y$ that to a point $z$ associates the unique point
    in $Y(z)$. This is a continuous function by semi continuity of $Y(z)$.
    \medskip \newline
    Denote $z_0, z_1 \in \partial B$ respectively the closest and farthest intersection of
    the line passing through $y$ and $x$. Observe that because $x \in C(Y)$,
    $\eta(z_1) \neq y$ and $\eta(z_0) = y$. Up to shrinking $B$, we find a
    neighbourhood $V \subset Y$ of $y$ containing $\eta(B)$, and a section
    $s : V \rightarrow NY$ such that for all $y' \in V$, the line passing through $y'$
    in direction $s(y')$ goes through $\partial B$, and in particular, the one at $y$
    passes through $z_0$ and $z_1$. We denote by $S \subset \partial B$ the set of these
    intersection points. From now on every topological operation happens in $S$.
    \medskip \newline
    Note that by our assumptions, $\eta$ is a homeomorphism on the hemisphere of $S$
    containing $z_0$ over its image. We denote the open hemisphere $U$.
    We now show that there is a point $z' \in S \backslash U$ such that $\eta(z') = y$.
    This will conclude the proof, as the only normal line to $y$ intersecting $S$ goes through
    $z_0$ and $z_1$, so $z' = z_1$ and this is a contradiction.
    \medskip \newline
    Assume there is no such point $z'$. Because $y \in \eta(U)$ is not in the image of
    $\eta_{\mid S \backslash U}$ and $\eta$ sends the cycle $\partial U$ to $\partial(\eta(U))$,
    this last cycle is not homologically trivial. But $\partial U = \partial (S \backslash U)$,
    so the cycle $\partial U$ is homologically trivial in $S \backslash U$, which contradicts
    what we found.
\end{proof}

\quad We now relate these special sets of points to differentiability of $\operatorname{dist}_Y$.
Using similar computations as in theorem \ref{thm:subdiffdist}, we can show,
because $\operatorname{dist}_Y$ and its square have the same differentiability points
out of $Y$, that outside of $Y \cup M_Y$ by Danskin's theorem, $\operatorname{dist}_Y$
is differentiable. In fact we have the following stronger statement.

\begin{prop}\label{prop:medialProp}
    The function $\operatorname{dist}_Y$ is $C^1$ on $\mathbb{R}^m \backslash (Y \cup \overline{M_Y})$.
\end{prop}

\begin{proof}
    Let $x \in \mathbb{R}^m \backslash (Y \cup \overline{M_Y})$. Let $U$ be a neighbourhood of
    $x$ contained in $Y \cup \overline{M_Y}$. Similarly to proposition \ref{prop:equalitySpecialPoints}
    we can then define a continuous function $\eta : U \rightarrow Y$ that to each $z \in U$ associates the unique
    $y \in Y(z)$. $\operatorname{dist}_Y$ is differentiable on $U$. Using the first part of
    Danskin's theorem and similar computations to theorem \ref{thm:subdiffdist}, we get
    \[ d_z \operatorname{dist}_Y = d_z g_{\eta(z)} \]
    From this, we deduce that $d \operatorname{dist}_Y$ is continuous on $U$.
\end{proof}

\subsection{Reducing to local continuous selections}

\quad In this section, $Y \subset \mathbb{R}^m$ will be a smooth submanifold, as $X$. 
Our goal will be to show that under certain assumptions, the distance function is
locally a continuous selection of finitely many $C^2$ functions, and so enters the Morse theory
framework developped in section \ref{sec:ContSelec}.
\medskip \newline
We'll denote by $\varphi$ the Riemannian exponential map restricted to $NY$, i.e for $(y, v) \in NY$,
$\varphi(y, v) = y + v$. The following proposition from \cite{AntonioMorseThEuclDist} uses
this map to find points where $\operatorname{dist}_Y$ is locally a continuous selection.

\begin{prop}
    Assume $Y$ is smooth and let $x \in \mathbb{R}^m$ be a regular value of $\varphi$.
    Then $Y(x) = \{ y_1, ..., y_n \}$ is finite. Moreover there exists
    \begin{itemize}
        \item $V$ a neighbourhood of $x$
        \item For all $i$, $W_i$ a neighbourhood of $y_i$ such that $\overline{W_i}$ are
        disjoint
    \end{itemize}
    that verify
    \[ \forall z \in V, \operatorname{dist}_Y(z) = \min_i \operatorname{dist}_{Y \cap \overline{W_i}}(z) \]
    and all $\operatorname{dist}_{Y \cap \overline{W_i}}$ are smooth on $V$.
\end{prop}

\begin{proof}
    We first prove that $Y(x)$ is finite. This set is compact. If we write $\pi : NY \rightarrow Y$
    the natural projection, we get $Y(x) \subset \pi(\varphi^{-1}(x))$. But note that $x$
    being a regular value, $\varphi^{-1}(x)$ is of codimension $m$ in $NY$, hence it is a set
    of disjoint points. The same applies after applying $\pi$, so $Y(x)$ is finite.
    \bigskip \newline
    We now construct the neighbourhoods : denote $Y(x) = \{y_1, ..., y_n\}$. For all $i$,
    because $x$ is regular, we choose a neighbourhood $U_i$ of $x - y_i$ in
    $\mathbb{R}^{m - \dim(Y)} \cong N_{y_i} Y$, $W_i$ a neighbourhood of $y_i$ in $\mathbb{R}^m$, and
    $V_i$ a neighbourhood of $x$ in $\mathbb{R}^m$ such that, up to trivialization,
    $(W_i \cap Y) \times U_i \subset NY$, and
    $\varphi : (W_i \cap Y) \times U_i \rightarrow V_i$ is a diffeomorphism.
    \medskip \newline
    Now note that for all $z$ in some neighbourhood $V \subset \bigcap_i V_i$ of $x$,
    $z - y$ is close to $x - y_i$ for all $y \in W_i \cap Y$. So if it happens that $z - y \in N_{y} Y$,
    then $z - y \in U_i$. From that observation, and the induced diffeomorphism, we deduce
    that there is only one such $y$. From that we get
    \[ \forall z \in V, \operatorname{dist}_{Y \cap \overline{W_i}}(z) = |\pi(\varphi_i^{-1}(z)) - z| \]
    where $\varphi_i := \varphi_{\mid (W_i \cap Y) \times U_i}$. So the distance is indeed
    a smooth function on $V$.
    \medskip \newline
    Lastly, by semi-continuity of $x \mapsto Y(x)$, we can assume that $V$ is small enough such
    that for $z \in V$, $Y(z) \subset \bigcup_i W_i$. Hence
    \[ \operatorname{dist}_Y(z) = \min_i \operatorname{dist}_{Y \cap \overline{W_i}}(z) \]
\end{proof}

\quad A natural question following this result would be to consider how restrictive is
the condition of being a regular value of $\varphi$, i.e what are these points,
is it likely that a critical point of ${\operatorname{dist}_Y}_{\mid X}$ is not a regular
value and what can we say in this case.
\medskip \newline
We can first observe that critical values of $\varphi$ are points in $\mathbb{R}^m$
verifying a degeneracy condition similar to the one defining the cut-locus $\operatorname{CLoc}(Y)$ :
$x$ is a critical value if there is $y \in Y(x)$ that is a degenerate minimum of the
function $y' \in Y \mapsto |x-y'|$. These points do not match $\operatorname{CLoc}(Y)$
though, as a regular value $x$ of $\varphi$ can be such that $Y(x)$ is not a singleton.
\medskip \newline
If we try to generalize the statement by dropping the regular value condition,
we can still show using the semi-continuity of the active
index and compactness that there is finitely many neighbourhoods $W_i$ such that for
$z$ close enough to $x$, $\operatorname{dist}_Y(z)$ can still be obtained by minimizing
over $\operatorname{dist}_{Y \cap \overline{W_i}}(z)$.\newline
We do not get smoothness of these distances near $x$ for arbitrary $Y$, a counterexample
is given by $Y = \mathbb{S}^{m-1}$ and $x = 0$ : if for some choice of $W_i$, the distances
were smooth, then $\partial_0 (\operatorname{dist}_{\mathbb{S}^{m-1} \cap \overline{W_i}})$ would
consist of a single point, which is not the case by theorem \ref{thm:subdiffdist}.
\medskip \newline
On the "likelyhood" side, it is shown that when $Y$ is algebraic, then
in the generic case, critical points of the distance do not intersect critical values of
$\varphi$ \cite{AntonioMorseThEuclDist}.

\section{Algebraic point of view on the euclidean distance}

\quad In this section, we still place ourselves in the euclidean space $\mathbb{R}^m$.
We will consider $X, Y \subset \mathbb{R}^m$ algebraic manifolds or varieties.
We still study the distance from $Y$ restricted to $X$, ${\operatorname{dist}_Y}_{\mid X}$.
Our goal will be to study the critical points of this function from an algebraic point of view.

\subsection{The critical point equation}

\quad We first introduce some notation : $Y$ is defined by the zero set of polynomials
$q_1, ..., q_n$, while $X$ is defined by the zero set of $p_1, ..., p_r$. We assume
that these are manifolds, which can be done without much trouble for what will come
by restricting to the smooth locus of $X$ and $Y$.
\bigskip \newline
From theorem \ref{thm:subdiffdist} proved in the last section, we now that a point $x \in X$ is critical for the distance function
iff there exists a number $k \geq 1$, $y_1, ..., y_k \in Y$ and $\lambda_1, ..., \lambda_k \in \mathbb{R}$
such that :
\begin{enumerate}
    \item The $y_i$ are all distinct
    \item $|x - y_i| = \operatorname{dist}_Y(x)$ for all $i$
    \item $\sum_i \lambda_i \frac{x - y_i}{|x - y_i|} \in N_x X$
    \item For all $i$, $\lambda_i \geq 0$ and $\sum_i \lambda_i = 1$.
\end{enumerate}
We want to transform these into algebraic equations, such that the solutions of these new
equations include true critical points of the distance function. The following equations
with unknowns $x, y_1, ..., y_k$ accomplish that

\begin{equation}\label{eq:KCP}\tag{KCP}
    \begin{array}{cl}
        \text{1.} & \text{The } y_i \text{ are all distinct}\\
        \text{2.} & \forall i, j, |x-y_i|^2 = |x-y_j|^2 \text{ and } x-y_i \in N_{y_i} Y\\
        \text{3.} & \text{The family } (x - y_i)_i \text{ is linearly dependent in } \mathbb{K}^m / N_x X
    \end{array}
\end{equation}

where $\mathbb{K} = \mathbb{R}$ for now.\newline
The first condition is an open one. The second one is algebraic because $N_{y_i} Y$
is spanned by $\nabla_{y_i} q_1, ..., \nabla_{y_i} q_n$, so $x-y_i \in N_{y_i} Y$ is equivalent
to asking that the family $\{x-y_i, \nabla_{y_i} q_1, ..., \nabla_{y_i} q_n\}$ is linearly
dependent. For similar reasons we can show that the last condition is also agebraic.\newline
It is also easy to check that true critical points provide a solution to these equations,
as for $y_i$ to minimize the distance from $x$ to $Y$, one needs $x-y_i \in N_{y_i} Y$.
\bigskip

\quad The next step to better understand these polynomials equations is to view them
as complex. For that, we consider the complexification of $X$ and $Y$ : $X_{\mathbb{C}}$
is the submanifold of $\mathbb{C}^m$ defined as the zero set of the complex polynomials
$p_1, ..., p_r$, and similarly for $Y_{\mathbb{C}}$. We also need to define what it means
for some complex vector to be in the normal bundle of $X_{\mathbb{C}}$ (or $Y_{\mathbb{C}}$) :
the normal bundle over $X_{\mathbb{C}}$ is defined as the vector bundle whose fiber at all
$x \in X_{\mathbb{C}}$ are
\[ N_x X_{\mathbb{C}} = \{ v \in \mathbb{C}^m \mid \forall w \in T_x X_{\mathbb{C}}, \sum_{i=1}^m v_i w_i = 0 \} \]
Note that the product involved is not an inner product on $\mathbb{C}^m$ but has the
advantage of being algebraic. It can be checked that this vector space is spanned
by the gradients of $p_1, ..., p_r$, so the observations made in the real case still hold.
With these definitions, we can then replace $\mathbb{K}$ with $\mathbb{C}$ in \eqref{eq:KCP},
and for what follows we will exclusively study this complex version. We will then
shorten $X_{\mathbb{C}}, Y_{\mathbb{C}}$ by $X, Y$, considering them as already complexified.
\bigskip

\quad In what follows, we will study particular cases of these equations and connect them to already existing notions
on algebraic varietes/manifolds (?). The case where $Y = \{y\}$ will lead to the notion
of \textit{euclidiean distance degree}, and when $k = 2$ and $X = \mathbb{C}^m$ we will get
the notion of \textit{bottleneck degree}.

\subsection{Euclidean distance degree}

\subsubsection{Definition and examples}

\quad As stated earlier, we consider the case where $Y = \{y\}$ and $X$ is some algebraic
submanifold of $\mathbb{C}^m$. \eqref{eq:KCP} then simplifies to
\[ x - y \in N_x X \]
where the unknown is $x$. This is equivalent to asking that $y \in x + N_x X$. We set
$\hat{N}_x X := x + N_x X$ and define the following bundle on $X$
\begin{equation}\label{eq:hatNX}
    \hat{N}X = \{ (x, v) \mid x \in X, v \in \hat{N}_x X \} \subset X \times \mathbb{C}^m
\end{equation}
This bundle has two natural projections $\pi_1 : \hat{N}X \rightarrow X$ and $\pi_2 : \hat{N}X \rightarrow \mathbb{C}^m$.
The set of solutions to \eqref{eq:KCP} is then $\pi_1(\pi_2^{-1}(y))$. Using the fact
that $\hat{N}X$ is of dimension $m$, one can show the following property.

\begin{prop}
    For generic $y \in \mathbb{C}^m$, the set $\pi_1(\pi_2^{-1}(y))$ is finite and it has
    a constant number of elements.
\end{prop}

The \textit{euclidean distance degree} (EDD) of $X$ is then defined as this generic number of solutions
to the critical point equation. It then gives an upper bound on the number of real solutions
to \eqref{eq:KCP}.
\medskip \newline
Consider the example where $X \subset \mathbb{C}^2$ is an ellipse. It is then defined by a
degree $2$ polynomial and we will show later that the euclidean distance degree of $X$ is $4$.
We see that for generic points $y$ in the bounded region of the plane $\mathbb{R}^2$ created
by $X$, we get the maximum amount, $4$ real critical points, and only $2$ when $y$ is in the
unbounded one.\newline
Not all submanifolds $X$ can have as many real critical point as the euclidean distance degree,
as can be seen with the parabola, of euclidean distance degree $4$ : for all points $y$,
we have at most $3$ real critical points.

\subsubsection{Chow ring and useful classes}

\quad We now present some tools that will help us compute the euclidean distance degree.
Let $X \subset \mathbb{P}^m$ be a reduced, irreducible complex projective variety (correct if this is wrong).
Let it be defined as the projective zero set of some polynomials, $p_1, ..., p_{m-n}$. $X$
is then of dimension $n$.

\begin{defi}
    The group of $k$-cycles of $X$, where $k \leq n$ is the free abelian group generated
    by the $Y \subset X$ that are closed, $k$-dimensional subvarieties denoted $Z_k(X)$.
\end{defi}

Consider $W \subset X$ a $k+1$-dimensional subvariety of $X$ and $f$ a rationnal function
over $W$. One can, for $Z \subset W$ a $k$-dimensional variety, define the order of vanishing
of $f$ along $Z$, denoted $\operatorname{ord}_Z(f)$ (cite) : intuitively it corresponds to
the exponent $c \in \mathbb{Z}$ such that near a generic point $x$ in $Z$, $f$ behaves
as some non zero factor times a degree $c$ monomial. One then defines the divisor of $f$
as
\[ \operatorname{div}(f) = \sum_{\substack{Z \subset W \\ \dim(Z) = k}} \operatorname{ord}_Z(f) Z \]
If we denote $R_k(X)$ the subgroup of $Z_k(X)$ generated by the divisors of rationnal
functions over some $k+1$ dimensional $W \subset X$, we get the following definition of
the Chow group.

\begin{defi}
    The Chow group of $X$ is defined as
    \[ A(X) = \bigoplus_k A_k(X) \]
    where for all $k$, $A_k(X) = Z_k(X)/R_k(X)$
\end{defi}

As an example, if $X = \mathbb{P}^2$ then, $[\{x^2 + y^2 - z^2 = 0\}] = 2 [\{x = 0\}]$
in $A(\mathbb{P}^2)$ because the following function, $f(x, y, z) = \frac{x^2 + y^2 - z^2}{x^2}$
is a rationnal function over a subvariety of $\mathbb{P}^2$ such that its divisor
gives exactly the difference of these two terms.
\medskip \newline
For our purpose, wich is computing the EDD, equivalently solving an intersection problem,
we will denote $A^k(X) := A_{n-k}(X)$, the classes of cycles of codimension $k$. We say that
two subvarieties, $Y, Z \subset X$ are generically transverse if for the generic intersection
smooth point, we indeed have transversality. The following proposition allows us to turn the
Chow group into the Chow ring with a structure compatible with intersection problems.

\begin{prop}[Chow ring]
    There is a unique multiplication making $A(X)$ into a ring such that for
    $Y, Z \subset X$ subvarieties that are generically transverse, we have
    \[ [Y] \cdot [Z] = [Y \cap Z] \]
    Moreover the multiplication respects the grading of $A(X)$ i.e it sends
    $A^i(X) \times A^j(X)$ onto $A^{i+j}(X)$ for all $i, j$.
\end{prop}

An important and motivating example is the case where $X = \mathbb{P}^m$ : the Chow ring is then
exactly $\mathbb{Z}[h] / (h^{n+1})$ where $h$ is the class of a hyperplane. A
subvariety $Y \subset \mathbb{P}^m$ of degree $d$ and codimension $k$ has the following
class $[Y] = d.h^c$. We can for example retrive Bezout's theorem by multiplying the classes
of two transversal curves. One can be more general and define a notion of degree for
certain elements in the Chow ring.

\begin{prop}[Degree of a class]
    There is a unique morphism, $\deg : A_0 (X) \rightarrow \mathbb{Z}$ such
    that $\deg([\p \cdot \p]) = 1$ and for all $Y, Z \subset X$ subvarieties of summed codimensions
    $n$, we have
    \[ |Y \cap Z| = \deg([Y][Z]) \]
\end{prop}

With the notion of the Chow ring, we can now introduce some useful classes that
live in the Chow ring of some variety. The first are the Chern classes.
\bigskip

\quad Consider a (complex) vector bundle over $X$, $E \rightarrow X$ of rank $r$ where $X$
is assumed to be smooth. We define the Chern classes of this bundle with the following
proposition.
\begin{prop}[Chern classes]
    There is a unique way to assign a class $c(E) = c_0(E) + ... + c_r(E) \in A(X)$, where
    $c_i(E) \in A^i(X)$ for all $i$, such that :
    \begin{enumerate}
        \item If $E$ is a line bundle, then $c(E) = 1 + c_1(E)$ and $c_1(E) = [\operatorname{div}(\tau)]$
        where $\tau$ is a rationnal section of $E$
        \item Let $s_1, ..., s_{r+k-1}$ be global sections of $E$. Their degeneracy locus
        is the variety $W \subset X$ consisting of the points where all of these sections are
        linearly dependant. If $\operatorname{codim}(W) = k$ (i.e the expected one), then $[W] = c_k(E)$.
        \item (Whitney's formula) If we have an exact sequence
        \[ 0 \rightarrow F \rightarrow G \rightarrow H \rightarrow 0\]
        of vector bundles over $X$, then $c(F)c(H) = c(G)$.
        \item (Functoriality) If $f : Y \rightarrow X$ is a morphism of smooth varieties,
        then $f^*(c(E)) = c(f^*E)$ where $f^* E$ is the pullback bundle of $E$ and
        \[ f^{*} : \begin{array}{cl}
            A(X) &\longrightarrow A(Y)\\
            \left[ V \right] &\longmapsto \left[ f^{-1}(V) \right]
        \end{array}
        \]
    \end{enumerate}
\end{prop}

\bigskip

\quad One other class of interest are polar classes. We define the embedded tangent bundle
of $X \subset \mathbb{P}^m$ as
\begin{equation}\label{eq:embeddedTX}
    \hat{T}X = \{ (x, v) \mid x \in X, v \in \ker (\nabla_x p_1 \mid ... \mid \nabla_x p_{m-n})\} \subset X \times \mathbb{C}^{m+1}
\end{equation}
Also denote for $x \in X$, $\mathbb{P}(\hat{T}_x X)$ the projective embedding of $\hat{T}_x X$.
Then for $L \subset \mathbb{P}^m$ a subspace, define the polar variety of $X$ with respect to $L$ as
\[ P(X, L) = \overline{\{ x \in X \mid \dim(\mathbb{P}(\hat{T}_x X) \cap L) > n - \operatorname{codim}(L) \}} \subset X \]
There is an intuitive example
when $L$ is a point and $X$ an ellipsoid : place yourself at point $L$, and consider all the
light rays arriving at your position. The light rays that form the "contour" of $X$ from
your perspective intersect $X$ exactly at $P(X, L)$.
\medskip \newline
For generic $L$ of codimension $n+2-k$, the class $[P(X, L)] \in A^k(X)$ is constant,
and we define the $k$-th polar class of $X$ as $p_k(X) = [P(X, L)]$. It turns out that
polar classes are Chern classes of some bundle, more precisely $c_k(\hat{T}^*X) = p_k(X)$,
where $\hat{T}^* X$ is the rank $n+1$ vector bundle whose fibers are dual to those of
$\hat{T}X$.
\medskip \newline
To see it, we write $L = \bigcap_i \ker \alpha_i$ with $\alpha_1, ..., \alpha_{n+2-k} \in (\mathbb{C}^{m+1})^*$.
For all $i$, the function
\[ \begin{array}{cl}
    X &\longrightarrow \hat{T}^*X\\
    x &\longmapsto {\alpha_i}_{\mid \hat{T}_x X}
\end{array} \]
is a section of $\hat{T}^*X$. The degeneracy locus of all these sections is the set $W$ of
all $x \in X$ such that ${\alpha_i}_{\mid \hat{T}_x X}$ are linearly dependant.
This is equivalent to asking that $L \cap \mathbb{P}(\hat{T}X)$ has dimension
greater than expected, i.e greater than $n - \operatorname{codim}(L)$. By condition $2.$,
$c_k(\hat{T}^*X) = [W] = p_k(X)$.
\bigskip

\quad Lastly, we present Segre classes and link them to the other ones. Consider again
a vector bundle $E \rightarrow X$ of rank $r$. We construct a projective bundle on
$X$ i.e whose fibers are projective spaces of dimension $r-1$,
$\mathbb{P}(E) \rightarrow X$, with projection map $p$, by projectifying the fiber over
each point $x \in X$. In this way, we have for all $x \in X$, $p^{-1}(x) = \mathbb{P}(E_x)$.
\medskip \newline
From this projective bundle, we can construct a new vector bundle, the tautological line bundle
over $\mathbb{P}(E)$
\[ \mathcal{O}_{\mathbb{P}(E)}(-1) = \{ ((x, L), w) \in \mathbb{P}(E) \times \mathbb{C}^{m+1} \mid w \in L \} \]
and $\mathcal{O}_{\mathbb{P}(E)}(1)$ its dual bundle.
\medskip \newline
Denote by $\xi = c_1(\mathcal{O}_{\mathbb{P}(E)}(1)) \in A^1(\mathbb{P}(E))$. We define
the $k$-th Segre class of $E$ for $k \in \llbracket 0, r \rrbracket$ as
\[ s_k(E) = p_{*}(\xi^{r-1+k}) \in A^k(X) \]
where the pushforward $p_*$ is defined as follows : $p$ must be a proper map. If this is the case,
$p_*$ is a linear map such that for $Y \subset X$ a subvariety, if $p(Y)$ has dimension strictly smaller than $Y$ then $p_*[Y] = 0$,
if not $p_*[Y] = d [p(Y)]$ where $d$ is the degree of the map $p_{\mid Y}$.
\newline
Again, we have relations to the other classes : Segre classes are inverses of Chern classes i.e
$s(E)c(E) = 1 \in A(X)$. Another important thing to note is that the bundle map $p$ embeds
$A(X)$ into $A(\mathbb{P}(E))$ through $p^*$. Moreover we have

\[ A(\mathbb{P}(E)) = A(X)[\xi] / (c_r(E) + c_{r-1}(E)\xi + ... + c_0(E)\xi^r) \]

\subsubsection{Computing the EDD}

\quad We now return to the topic of computing the EDD of a smooth complexified algebraic
manifold $X \subset \mathbb{C}^m$, defined by polynomials $p_1, ..., p_{m-n}$.
Our tools for solving intersection problems requiring
projective manifolds, we will take the "projective closure" of $X$ and its bundles
of interest.
\medskip \newline
For that we consider the canonical embedding (we will discuss this choice later on)
$\mathbb{C}^m \hookrightarrow \mathbb{P}^m$. The hyperplane at infinity $H_{\infty}$ is the set $\mathbb{P}^m \backslash \mathbb{C}^m$.
The isotopic quadric at infinity is defined as
\[ Q_{\infty} = \{ [x_1 : ... : x_n : 0] \in H_{\infty} \mid x_1^2 + ... + x_n^2 = 0\} \]
Using the canonical embedding, $X$ is embedded in its projective closure $\overline{X}$,
and we denote $X_{\infty} = \overline{X} \backslash X$.
\medskip \newline
Embedding $\hat{N}X$ as defined in \eqref{eq:hatNX} requires a bit more care and is done properly in
(ref section) following (cite). For that we need $X$ the be orthogonally general which means
that $\overline{X}$ is smooth, $\overline{X}$ is tranvsersal to $H_{\infty}$ and $X_{\infty}$ is transversal to
$Q_{\infty}$. Denote $\overline{\hat{N}X}$ the projective closure in
$\overline{X} \times \mathbb{P}^m$ of $\hat{N}X$. In this situation, this is a projective
bundle, and we can define a vector bundle $\hat{N} \overline{X}$
over $\overline{X}$ that we call the euclidean normal bundle such that
$\overline{\hat{N}X} = \mathbb{P}(\hat{N}\overline{X})$. We have the following maps

\begin{center}
\begin{tikzcd}
    \mathbb{P}(\hat{N}\overline{X}) \arrow[rr, "\iota \p=\p \operatorname{id}_{\overline{X}} \times \pi_2"] \arrow[dr, "\pi_1"']
    & & \mathbb{P}(\overline{X} \times \mathbb{C}^{m+1}) \arrow[dl, "\pi'_1"]\\
    & \overline{X}
\end{tikzcd}
\end{center}

\quad We now compute the EDD of $X$. It was defined as the number of elements of
$\pi_1(\pi_2^{-1}(y)) \subset \mathbb{C}^m \hookrightarrow \mathbb{P}^m$ for generic $y$.
To find this number we use the $\deg : A(\mathbb{P}^m) \rightarrow \mathbb{Z}$ map, the
degree of a finite set being its size. We have
\begin{align*}
    \operatorname{EDD}(X) &= \deg([\pi_1(\pi_2^{-1}(y))])\\
    &= \deg([\pi_1(\iota^{-1}(\overline{X} \times \{y\}))])
\end{align*}

Denote $\xi = c_1(\mathcal{O}_{\mathbb{P}(\overline{X} \times \mathbb{C}^{m+1})}(1))$. It can
be shown that $\xi = [\overline{X} \times H]$, where $H$ is any hyperplane of $\mathbb{P}^m$,
using property 1. of Chern classes. Because $H$ can be any hyperplane, one has
$\xi^m = [\overline{X} \times \{y\}]$.\newline
We also define $\eta = c_1(\mathcal{O}_{\mathbb{P}(\hat{N}\overline{X})}(1))$, and property
4. of Chern classes gives us $\iota^*(\xi) = \eta$. Rewriting what we had, because $\pi_1$
is of degree $1$, we obtain
\begin{align*}
    \operatorname{EDD}(X) &= \deg((\pi_1)_* \iota^*(\xi^m)))\\
    &= \deg((\pi_1)_* \eta^m)\\
    &= \deg(s_n(\hat{N}\overline{X}))
\end{align*}
by definition of Segre classes. To finish, we need some more information about the euclidean
normal bundle.
\medskip \newline
The tautological bundle over $\overline{X}$ is the bundle $\mathcal{O}_{\overline{X}}(-1)$
whose fiber at $[x] \in \overline{X}$ is the line $\mathbb{C}x \subset \mathbb{C}^{m+1}$.
The normal bundle of $\overline{X}$, $\mathcal{N}_{\overline{X}}$ is a bundle verifying
the following exact sequence

\begin{center}
    \begin{tikzcd}
        0 \arrow[r] & \hat{T}\overline{X} \arrow[r] & \overline{X} \times \mathbb{C}^{m+1} \arrow[r]
        & \mathcal{N}_{\overline{X}} \arrow[r] & 0
    \end{tikzcd}
\end{center}

where $\hat{T} \overline{X}$ is defined in \eqref{eq:embeddedTX}. By (cite), we
have $\hat{N}\overline{X} \simeq \mathcal{N}^*_{\overline{X}} \oplus \mathcal{O}_{\overline{X}}(-1)$.
\medskip \newline
Property 3. of Chern classes and the fact that Segre classes are their inverses yields
$s(\hat{N}\overline{X}) = s(\mathcal{N}^*_{\overline{X}})s(\mathcal{O}_{\overline{X}}(-1))$.
By dualizing the exact sequence, and applying property 3. of Chern classes again, we get
\[ s(\mathcal{N}^*_{\overline{X}})s(\hat{T}^* \overline{X}) = s(\overline{X} \times \mathbb{C}^{m+1}) \]
For all $k \neq n$, $s_k(\overline{X} \times \mathbb{C}^{m+1}) = 0$, because $\xi^{m+1} = 0$.
So $s(\overline{X} \times \mathbb{C}^{m+1}) = 1$, and we get
$s(\mathcal{N}^*_{\overline{X}}) = c(\hat{T}^* \overline{X})$.\newline
Lastly, by property 1., one can compute
\[ c(\mathcal{O}_{\overline{X}}(-1)) = 1 - c_1(\mathcal{O}_{\overline{X}}(1)) = 1 - \xi \]
and deduce
\[ s(\mathcal{O}_{\overline{X}}(-1)) = \sum_{k=0}^m \xi^k \]
We then get
\begin{align*}
    s_n(\hat{N}\overline{X}) &= (c(\hat{T}^*\overline{X}) s(\mathcal{O}_{\overline{X}}(-1)))_n\\
    &= \sum_{k=0}^n c_k(\hat{T}^*\overline{X}) \xi^{n-k}\\
    &= \sum_{k=0}^n p_k(X) \xi^{n-k}
\end{align*}
so we conclude
\[ \operatorname{EDD}(X) = \sum_{k=0}^n \deg(p_k(X)) \]
After computing the polar classes we get the following result
\begin{theorem}
    If $X \subset \mathbb{C}^m$ is a smooth submanifold defined by polynomials (+ hypothesis ?) $p_1, ..., p_{m-n}$ of degrees
    $d_1 \leq ... \leq d_{m-n}$, then
    \[ \operatorname{EDD}(X) = d_1 ... d_{m-n} \sum_{i_1 + ... + i_{m-n} \leq n+1} (d_{i_1} - 1)^{i_1} ... (d_{i_{m-n} - 1})^{i_{m-n}} \]
\end{theorem}
